Part 1: HTML

All web pages are written in HTML, which stands for Hypertext Markup Language. A markup language is a system for annotating text that indicates to a computer how the text should be processed. Specifically, HTML tells a web browser how different content on a web page should be displayed.

HTML consists of tags, which are essentially labels for content. Tags are made up of keywords surrounded by angle brackets <>. For example, <h1> is a tag which indicates when text is a heading. Most tags, such as the <h1> tag, come in pairs that encapsulate the relevant content. You can think of HTML as similar to a set of parenthesis. After this opening tag, fill in the content “inside” the tag, and close it with a "closing tag" (kind of like how you "close" a set of parenthesis), which is same except with a forward slash before the keyword: </h1>. A pair of such tags and the content it describes is together called an HTML element. Thus:

<h1>This is a heading.</h1>

Part 3: jQuery Now that you’ve learned how to create, format, and style content on a webpage, it’s time to learn how to add functionality and make an actual web *application*. To do this you use Javascript, a programming language built for use with websites. Javascript lets you manipulate the content on the webpage, and in our case, communicate with the server that stores your application’s data (we’ll get to this in Part 4).

Manipulating HTML data is also called DOM manipulation, and to make it easier for you we’ve included a “library” of extra Javascript called jQuery in Chirp that you can use in your own code. jQuery’s slogan is “write less, do more”, and will be helpful for you in trying to prototype your application in as little time as possible.

jQuery basics: Selecting HTML - one of the big benefits of jQuery is the ease with which you can use it to select which HTML elements you want to modify. To reference a component of your HTML, you use a dollar sign, followed by parentheses surrounding quoted text specifying an HTML id or a class you want to reference. To reference an id, you quote a “hashtag” followed by the id, and to reference a class you quote a period followed by the class. Examples: $(“#messages”) // references element with id “messages” $(“.buttons”) // references all elements with class “buttons”

Modifying selected HTML - Once you have selected HTML, you need to be able to manipulate it. Manipulations of HTML include changing its contents, moving things around, adding/removing elements, and changing its CSS. In jQuery, these manipulations take the form of “functions” that are appended to the HTML “selector” with a period between them. A basic example might look like this:

Suppose you have some HTML like below: <p id='dinosaur-paragraph'> Nothing here right now! </p> As you should know at this point, this will display a paragraph that says "Nothing here right now!", with the id "dinosaur-paragraph". It's also possible there might be some CSS applied to that paragraph, but that doesn't matter right now. Now, the following jQuery will change the text of the paragraph: $("#dinosaur-paragraph").html("Dinos are cool!"); Here's how this statement works bit-by-bit: $("#dinosaur-paragraph") Selects the HTML element with the id "dinosaur-paragraph" .html("Dinos are cool!"); Changes the HTML between the tags of the element specified in the previous part (in this case "dinosaur-paragraph") to the text specified between the parentheses. In this case, the text is "Dinos are cool!". Keep in mind, quotes are mandatory. Here's a table of some basic jQuery manipulations: Function Description $("#example").html("text here"); Changes content of tag "example" to "text here". $("#example").append("text to append"); Appends "text to append" to the content of tag "example". $("#example").prepend("text to prepend"); Prepends "text to prepend" to the content of tag "example". $("#example").toggle(); Makes element "example" visible if it's currently invisible, or vice-versa. $("#example").css("property", "value"); Modifies the CSS that applies to id "example", setting the value of "property" to "value". Look in main.js of Chirp and you can see some of these jQuery functions being used. Suggested Exercise: Modify Chirp so that messages are displayed in the reverse order. Getting input with jQuery - So far, we can change how we display things by using jQuery functions. The next step is to get input from the user. For the purposes of the example, we're going to be looking at two types of input - text and button. To get the value of a text box, simply use a selector that selects the text box with the function .val(). However, to make use of this, there is one more concept to learn: variables. You define a variable with the keyword "var", then a space, then the name of the variable. Use an equals sign after it to set the value of the variable. Example: var myInput = $("#input").val(); The above statement makes myInput represent the value of the text box with the id "input" at the time that function was used. Then, you can use the variable as a value for other functions. Example: $("#what-did-i-input").html(myInput); This will set the text of the element with id "what-did-i-input" to the text you inputted. Keep in mind, you don't use quotes with your variable name (otherwise it will literally display as the word "myInput").
